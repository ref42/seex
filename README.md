# SeEx

Seek & Export.

`SeEx` is distributed here as binary releases for clipboard-based LCSC ID tracking,
BOM CSV export, and EDA library export through `npnp`.

## Downloads

Download the latest release from:

https://github.com/ref42/seex/releases

## Scope

This public repository is for binary distribution, issue tracking, and release notes.

Source code is not publicly distributed from this repository.

Issues with clear reproduction details are welcome. External pull requests are not merged.
